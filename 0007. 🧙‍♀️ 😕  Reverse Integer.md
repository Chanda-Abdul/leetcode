Given a signed 32-bit integer `x`, return `x` with its digits reversed. If reversing `x` causes the value to go outside the signed 32-bit integer range `[-2Â³Â¹, 2Â³Â¹ - 1]`, then return `0`.

<b>Assume the environment does not allow you to store 64-bit integers (signed or unsigned).</b>

## Example 1:
````
Input: x = 123
Output: 321
````
## Example 2:
````
Input: x = -123
Output: -321
````
## Example 3:
````js
Input: x = 120
Output: 21
````
## Example 4:
````js
Input: x = 0
Output: 0
````

## Constraints:
- `-2Â³Â¹ <= x <= 2Â³Â¹ - 1`

## Solution 
````ts
function reverse(x) {
  let reverse = Math.abs(x).toString().split('').reverse().join('');

  //handle value that are outside the signed 32-bit integer range
  if (reverse > Math.pow(2, 31)) {
    return 0;
  }

  //handle negative numbers
  return x < 0 ? -reverse : reverse;
}

reverse(-123); //-321
reverse(120); //021
reverse(0); //0
````

## Time & Space Complexity
- Time Complexity => `O(n)`
- Space Complexity => `0(1)`
